## What is NaowhQOL?

NaowhQOL is a performance and gameplay-focused addon designed to improve your World of Warcraft experience by enhancing system performance, gameplay clarity, and player awareness ‚Äî without changing your UI or adding visual clutter.

This is not a UI pack.
This is not a visual overhaul.
This is a modular Quality of Life system.

## Features

### Combat

* Combat Timer
* Combat Alerts
* Combat Logger
* GCD Tracker

### HUD

* Mouse Ring
* Crosshair
* Focus Cast Bar
* Dragonriding HUD

### Tracking

* Buff Monitor
* Consumables Tracker
* Stealth / Stance Detection
* Range Check

### Reminders

* Talent Reminder
* Emote Detection

### System

* Smart performance presets
* Automatic graphic optimization
* FPS stability system
* Network smoothing (latency stability)
* CVAR optimization engine
* Dynamic texture & shadow control
* Profile import/export system
* Slash custom commands
* Advanced optimizations
* Misc utilities
