A lightweight Quality of Life addon for World of Warcraft focused on performance, clarity and smooth gameplay.

> Optimize your game. Not your UI.

## What is NaowhQOL?

NaowhQOL is a performance and gameplay-focused addon designed to improve your World of Warcraft experience by enhancing system performance, gameplay clarity, and player awareness without changing your UI or adding visual clutter.

This is not a UI pack.
This is not a visual overhaul.
This is a modular Quality of Life system.

## Features

### Combat

* Combat Timer
* Combat Alerts
* Combat Logger
* GCD Tracker

### HUD

* Mouse Ring
* Crosshair
* Focus Cast Bar
* Dragonriding HUD

### Tracking

* Buff Monitor
* Consumables Tracker
* Stealth / Stance Detection
* Range Check

### Reminders

* Talent Reminder
* Emote Detection

### System

* Smart performance presets
* Automatic graphic optimization
* FPS stability system
* Network smoothing (latency stability)
* CVAR optimization engine
* Dynamic texture & shadow control
* Profile import/export system
* Slash custom commands
* Advanced optimizations
* Misc utilities

## Why NaowhQOL?

Because most addons focus on visuals.
NaowhQOL focuses on **performance, clarity, and gameplay stability**.

It helps you:

* Get more stable FPS
* Reduce stuttering
* Improve responsiveness
* Increase combat awareness
* Reduce visual noise
* Play smoother in high-pressure content

> Clean gameplay. Stable performance. Competitive clarity.

## Installation

1. Download the latest release
2. Rename it to NaowhQOL
3. Extract into:

```
World of Warcraft/_retail_/Interface/AddOns/
```

4. Restart the game
5. Configure using `/nao`
